Internet Access Instructions

Follow these steps to connect to the internet

Wi-Fi Connection Setup

1. Locate the Wi-Fi icon in the system tray (bottom-right corner on Windows, top-right corner on Mac).
2. Click on the Wi-Fi icon to view available networks.
3. Select the network named: [Your Network Name]
4. When prompted, enter the password: [Your Password]
5. Click "Connect" and wait for the connection to establish.
6. Once connected, you should see "Connected" next to the network name.
7. Open a web browser to test your connection.

Troubleshooting Tips

• If you can't see the network, make sure Wi-Fi is enabled on your device.
• If the password doesn't work, check for correct capitalization and special characters.
• Try moving closer to the router if the signal is weak.
• Restart your device if connection problems persist.
• Contact IT support at: [Support Phone/Email]

Ethernet Connection Setup

1. Connect one end of the Ethernet cable to your computer.
2. Connect the other end to an available Ethernet port on the wall or router.
3. Wait for the connection to be established (usually indicated by a light on the Ethernet port).
4. Open a web browser to test your connection.

Important Notes

• Do not share the Wi-Fi password with unauthorized users.
• For security reasons, the password may be changed periodically.
• Report any suspicious network activity immediately.
